# BreadLab

Помощник домашнего пекаря: расчёты, рецепты и документация выпечки.

Проект состоит из двух частей:
*   **Клиентская** (VK-бот + Redis)
*   **Серверная** (Django + PostgreSQL)

Взаимодействуют через HTTP API. Основной функционал покрыт тестами.

---

## VK-бот. Функционал

*   **Калькулятор закваски:** пересчёт опары с 50% на 100% влажность и обратно по заданным пропорциям.
*   **Калькулятор пропорций:** умножение ингредиентов в рецепте на нужное количество порций.
*   **Загрузка рецептов:** распознавание рецептов по тексту и фото с помощью LLM-модели.
*   **Редактирование рецептов:** правки на естественном языке через LLM (например, "добавить 5г соли").
*   **Управление рецептами:** создание, редактирование, удаление. Версионирование -- сохранение как новая версия или обновление существующего рецепта.

## Стек

*   Python 3.12+
*   vkbottle (VK API)
*   python-statemachine (FSM)
*   Redis (хранение состояний и промежуточных данных пользователя)
*   pytest
*   Docker / docker-compose

## Локальный запуск

1.  **Клонируйте репозиторий**
    ```bash
    git clone https://github.com/aklukyanov/BreadLabBot_vk.git
    cd BreadLabBot_vk
    ```

2.  **Создайте файл .env в корне проекта:**
    ```env
    VK_TOKEN=ваш_токен
    ```

3.  **Запустите через Docker**
    ```bash
    docker-compose up --build
    ```

Серверная часть проекта: https://github.com/aklukyanov/BreadLab-server
